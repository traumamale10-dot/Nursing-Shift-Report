✅ บันทึกแล้ว
📌 วิธีตั้งค่า:
1. ไปที่ script.google.com → สร้าง Project ใหม่
2. วางโค้ดจากไฟล์ setup_sheet.gs ที่ได้รับ

4. Deploy → New Deployment → Web App → Execute as Me → Anyone
5. คัดลอก URL มาวางด้านบน
6. กดบันทึก และทดสอบ
